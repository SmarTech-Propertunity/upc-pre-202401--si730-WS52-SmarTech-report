*Tabla de épicas establecidas para las historias de usuarios.*

| User<br>Story<br>ID | Título | Descripción | Criterios de Aceptación | Relacionado<br>con Epic |
|---------------------|--------|-------------|-------------------------|-------------------------|
| US01 | Esquematización y programación de una landing page | **Como** usuario general de la aplicación web, **quiero** que haya una página sencilla y llamativa como presentación, **para** que pueda revisar varios detalles de la aplicación, como sus funcionalidades, las suscripciones, información sobre la startup, etc. | **Dado que** tengo el link de la página web copiado en mi portapapeles, **cuando** pegué el link en la barra de búsqueda y presione siguiente, **entonces** podré revisar la landing page de la aplicación web Propertunity y podre revisar sus funcionalidades, suscripciones y más información útil. | Epic 01 |
| US02 | Integración de búsqueda en un Web Browser | **Como** usuario general de la aplicación web, **quiero** ser capaz de buscar la landing page de la aplicación web directamente desde el browser con el uso de palabras clave, **para** que mi búsqueda sea más fácil y no tener que copiar y pegar el link de la página en todo momento que quiera acceder. | **Dado que** estoy en el navegador web de mi dispositivo, **cuando** ponga palabras clave relacionadas directamente con Propertunity o con el rubro inmobiliario peruano, **entonces** me saldra la landing page de Propertunity como resultado en cualquier sección de la primera página por segunda página de resultados. | Epic 01 |
| US03 | Integración de links para redes sociales en el Footer | **Como** usuario general de la aplicación web, **quiero** ser capaz de acceder a las principales redes sociales de la startup que diseña Propertunity, **para** poder revisar información actualizada sobre sus planes de negocio, eventos con su público o nuevas actualizaciones en Propertunity o en algún otro producto web que ofrezcan. | **Dado que** me encuentro dentro de landing page de Propertunity, **cuando** me dirija a la sección de la base o footer de la landing page y le de click a alguno de los simbolos de redes sociales que tiene la startup de SmarTech, los cuales son de Facebook, X, Instagram y LinkedIn, **entonces** podré acceder directamente al perfil de la startup en la red social que elegí. | Epic 01 |
| US04 | | | | |
| US05 | | | | |
| US06 | Preparar una interfaz llamativa y dinámica | **Como** usuario general de la aplicación web, **quiero** poder interactuar con la página correctamente, que muestre una interfaz llamativa y que sea dinámica, **para** que mi experiencia (y primera impresión) me motive a usar el producto y/o servicio que se ofrece en la página. | **Dado que** estoy visualizando e interactuando el landing page de Propertunity, **cuando** quiera tener una primera impresión satisfactoria, **entonces** podré apreciar un landing page vistoso y atrayente a primera vista. | Epic 02 |
| US07 | Incorporar una lista con las principales características | **Como** usuario general de la aplicación web, **quiero** saber las principales características que me ofrece el landing page ensalsando las virtudes y/o funcionalidades diferentes a la competencia, **para** poder diferenciar a Propertunity de la competencia. | **Dado que** quiero ver las principales características y funcionalidades del producto ofrecido, **cuando** esté dentro del landing page, **entonces** podré apreciar una lista o colección mostrando las principales características, funcionalidades, diferencias ante la competencia, valores agregados de Propertunity. | Epic 02 |
| US08 | Evidenciar casos de uso real | **Como** usuario general de la aplicación web, **quiero** poder ver o visualizar indicios de otros usuarios quienes ya usaron el producto, **para** poder tener la certeza y garantía de que lo que promocionan es un producto funcional y de confianza. | **Dado que** quiero poder ver los casos, testimonios, comentarios o reseñas de otros usuarios quienes ya usaron el producto, **cuando** esté dentro del landing page y vea el apartado de 'testimonios', **entonces** podré ver de manera detallada los testimonios o experiencias de otros usuarios. | Epic 02 |
| US09 | Clasificar las funcionalidades | **Como** usuario general de la aplicación web, **quiero** poder ver un apartado donde se encuentre clasificadamente las funcionalidades del landing page, **para** poder explorar de mejor manera lo que se ofrece. | **Dado que** quiero ver todas las funciones y características del landing page, **cuando** me dirija al apartado de funciones, **entonces** podré navegar más agusto y ordenadamente dentro del landing page conociendo de esta forma mejor el producto. | Epic 02 |
| US010 | Implementación de una barra de búsquedas | **Como** usuario general de la aplicación web, **quiero** poder buscar dentro del landing page las funciones tanto que se ofrecen del landing page como las funciones del producto, **para** poder familiarizarme mejor con el producto y entender su funcionamiento. | **Dado que** desea buscar las funciones, características a base de palabras claves, **cuando** se dirija a la barra de búsquedas e inserte la palabra clave de lo que necesita, **entonces** se mostrará todas las coincidencias posibles para que el usuario pueda informarse mejor y conocer el producto. | Epic 02 |
